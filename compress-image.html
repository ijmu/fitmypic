What makes an image file smaller

Image file size is affected by pixel dimensions, format, and quality. Reducing width and height often makes the biggest difference. Lowering JPG or WebP quality can also reduce file size while keeping the image visually acceptable.

Which format should you choose

  • JPG is good for photos and product images without transparency.
  • PNG is good for logos, screenshots, icons, and transparent graphics.
  • WebP is good for websites because it often creates smaller files.

Quality setting tips

A quality setting around 80% is a good starting point for JPG and WebP. If the image still looks clear, try 70%. If text or edges look rough, increase the quality or use PNG for graphic-style images.
